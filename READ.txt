# FlappyBat

[![License](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)

*FlappyBat* é uma versão divertida e minimalista do famoso jogo Flappy Bird, onde você controla um morcego voando através de obstáculos. Este projeto foi criado como trabalho da faculdade e foi feito em python.

![FlappyBat Screenshot](screenshot.png)

## Como Jogar

1. Clone ou faça o download deste repositório para a sua máquina.

2. Abra o arquivo `FlappyBat.py` em uma IDE como Pycharme por exemplo.

3. Instale a biblioteca 'pygame'

4. Execute o arquivo 'manage.py'

5. Pressione a barra de espaço para fazer o morcego voar e evite colidir com os obstáculos.

6. Tente conseguir a maior pontuação possível!

## Recursos

- Desenvolvido em Python.
- Jogo simples e viciante para todas as idades.
- Faça o download e jogue offline.
